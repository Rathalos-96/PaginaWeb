# PaginaWeb
PaginaWeb

## Cómo ejecutar

1. Abrir `index.html` en un navegador moderno.
2. Verificar que el CSS se cargue correctamente desde `css/styles.css`.
3. La página es responsive y contiene secciones de información de SQL Server con imágenes locales en `img/`.
4. Para cambiar imágenes, reemplazar archivos en `img/` con los mismos nombres (`hero.jpg`, `server.jpg`, `monitor.jpg`).
